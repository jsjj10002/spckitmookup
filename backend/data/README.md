# 데이터베이스 파일

PC 부품 데이터베이스 관련 파일을 저장하는 디렉토리입니다.

## 파일 목록

- `pc_data_dump.sql`: PC 부품 데이터베이스 덤프 파일 (11MB)
- `PC 부품 DB 스키마 가이드.pdf`: 데이터베이스 스키마 가이드 문서

## 사용 목적

### RAG (Retrieval-Augmented Generation) 구현
이 데이터베이스를 활용하여:
- 벡터 데이터베이스에 임베딩 저장
- 사용자 쿼리에 대한 정확한 부품 검색
- 실제 데이터 기반 PC 부품 추천

### 데이터 구조
데이터베이스에는 다음 테이블들이 포함되어 있습니다:
- `cpu`: CPU 부품 정보
- `gpu`: GPU 부품 정보
- `memory`: 메모리 부품 정보
- `motherboard`: 메인보드 부품 정보
- `storage`: 저장장치 부품 정보
- `psu`: 파워서플라이 부품 정보
- `case`: 케이스 부품 정보
- `cpu_cooler`: CPU 쿨러 부품 정보
- 기타 호환성 테이블들

## 향후 작업

1. **데이터베이스 구축**
   - PostgreSQL 또는 MySQL에 데이터베이스 생성
   - SQL 덤프 파일을 데이터베이스에 임포트

2. **벡터 데이터베이스 구축**
   - 부품 정보를 벡터로 임베딩
   - Pinecone, Weaviate, Chroma 등 벡터 DB에 저장

3. **RAG 검색 API 개발**
   - 사용자 쿼리를 벡터로 변환
   - 유사한 부품 검색
   - Gemini API에 컨텍스트로 제공

## 주의사항

- SQL 파일은 11MB로 큰 파일입니다
- Git에 커밋하지 않도록 `.gitignore`에 추가 고려
- 프로덕션 환경에서는 데이터베이스 서버를 별도로 운영
